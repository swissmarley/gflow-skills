description: Create article illustrations, cover art, photo essays, and visual narratives with art-directed AI photography
---

Run the editorial-design skill. Capture the editorial brief (article topic, tone, publication style, visual references), run `gflow doctor`, define a shot list (hero, supporting, detail, atmosphere), set art direction per shot (lighting, angle, color treatment, concept), generate with `gflow image --model imagen-4 --outputs 4` per shot, curate the best selects and upscale heroes with `--upscale 4k`, generate motion pieces with `gflow video` for digital editorial, carry the brand with `--project <brand>`, and deliver the image set plus art-direction notes and layout suggestions after running the full quality gate.
