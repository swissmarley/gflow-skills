description: Produce photorealistic architectural renders, walkthrough videos, before/after transformations, and real estate marketing
---

Run the architectural-viz skill. Capture the project brief (building type, style, location, time of day), run `gflow doctor`, generate exterior establishing shots with `gflow image --model imagen-4 --ratio 16:9`, generate interior scenes with a consistent design language, create a walkthrough with `gflow video --model veo-3.1-quality --duration 15`, build before/after transformations with `gflow edit image` on base renders, upscale presentation shots with `--upscale 4k`, carry the brand with `--project <brand>`, and deliver the render set plus walkthrough video plus presentation deck assets after running the full quality gate.
