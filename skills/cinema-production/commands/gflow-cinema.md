Run the cinema-production skill. Break the script into scenes, characters, locations, and mood, run `gflow doctor`, storyboard key frames per scene with `gflow image`, get user approval on the storyboard, then render per-scene video with `gflow video --model veo-3.1-quality`, hold character continuity with `--character <name>` and brand with `--project <brand>`, build transitions with `gflow video --start-frame --end-frame`, batch-render the full sequence with `gflow batch`, assemble the clips into one continuous film with `gflow extend --media-id <opening clip>` plus repeated `--prompt "<what happens next>"` per beat and `--add-clip` to append rendered clips (continue an existing timeline with `--scene <id>` from `gflow scene list`), and deliver numbered clips, the combined scene video, an assembly guide, and the storyboard doc after running the full quality gate. Execute every gflow command with one shared `--project <name>` for the run (the brand project if one exists) and `--no-headed`; only `gflow auth login` runs headed.
