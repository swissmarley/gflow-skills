---
description: Create concept art, environment painting, character design, cutscene video, UI mockups, and marketing key art for games
---

Run the game-assets skill. Capture the game brief (genre, art style, platform, reference games or films), run `gflow doctor`, explore concept art wide with `gflow image --outputs 8`, lock the art direction from user selects, paint key environments at different times and weather, create characters with `gflow character create` for consistency, generate cutscenes with `gflow video` using character references, build UI mockups for menus, HUD, and inventory, upscale marketing key art with `--upscale 4k`, carry characters with `--character <name>` and style with `--project <brand>`, and deliver the asset set after running the full quality gate.
